# MoiKanakku (மொய் கணக்கு)

A free & offline Tamil–English MOI (மொய்) management system for South Indian weddings and family functions.

## Features
- Tamil & English UI
- One Excel file per function (date + name)
- Autosuggest for names and villages
- Total collection
- Edit last 5 records
- Offline & safe

## Installation (Windows)
1. Download from GitHub Releases
2. Extract ZIP
3. Run MoiKanakku.exe
